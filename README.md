# 🐦 Fink — dein lokaler Sprach-Agent

> **Sprich, statt zu tippen.** Fink verwandelt deine Stimme in sauberen Text —
> in jedem Programm, in dem du schreibst. Und es kann noch einiges mehr.

Fink läuft als kleines Symbol in deiner Taskleiste. Du hältst eine Taste gedrückt,
sprichst — und beim Loslassen steht der fertige, aufgeräumte Text dort, wo dein
Cursor ist: in Word, im Browser, in E-Mails, im Chat, überall.

---

## Was Fink kann

- **🎙️ Diktieren** — Taste halten, sprechen, loslassen. Fink erkennt deine Sprache,
  bereinigt Füllwörter und Satzzeichen und fügt den Text ins aktive Fenster ein.
- **✍️ Text bearbeiten per Sprache** — markieren und sagen „kürzer", „förmlicher",
  „auf Englisch" — Fink schreibt die Stelle um.
- **🤖 Fragen & Recherche** — stell Fink eine Frage; bei Bedarf recherchiert es
  selbst im Web und antwortet in einem schwebenden Fenster.
- **✅ To-Dos & Listen**, **⏱️ Zeiterfassung mit Abrechnung**, **📝 Notizen** (Markdown),
  **🎬 Teleprompter** (scrollt beim Vorlesen automatisch mit) — alles eingebaut.
- **🔊 Sprachausgabe** — Fink kann Antworten auch vorlesen.
- **🧩 Eigene Modi** — leg dir selbst „Modi" an (Ton, Stil, Format) für jede
  Situation: lockere Notiz, förmliche Mail, Programmier-Kommentar …

## Privat & flexibel

Fink läuft **komplett lokal** auf deinem Rechner — deine Stimme und deine Texte
müssen das Gerät **nicht verlassen**. Wer mehr Leistung möchte, kann optional
Cloud-KI (z. B. ChatGPT, Claude, Gemini) per eigenem Schlüssel anbinden. Du
entscheidest. Nur die Web-Recherche und bewusst gewählte Cloud-Dienste gehen
online.

## Installation

Lade unten unter **Releases** das passende Paket herunter:

| System | Datei | Hinweis |
|---|---|---|
| **Windows** (ohne NVIDIA-GPU) | `Fink-windows.zip` | läuft überall |
| **Windows** (mit NVIDIA-GPU) | `Fink-windows-cuda.zip` | nutzt die GPU → schneller |
| **macOS** | `Fink-macos.zip` | Apple Silicon empfohlen |
| **Linux** | `Fink-linux.tar.gz` | |

**So geht's (Windows):**
1. `Fink-windows.zip` herunterladen und **entpacken** (Rechtsklick → „Alle extrahieren").
2. Im entpackten Ordner **`Fink.exe`** starten.
3. Beim ersten Start meldet Windows evtl. „Computer geschützt" (die App ist noch
   nicht signiert) → **„Weitere Informationen" → „Trotzdem ausführen"**.
4. Ein kurzer Einrichtungs-Assistent führt dich durch Mikrofon & Co.

> Fink öffnet **kein** großes Fenster — es lebt als **Symbol unten rechts in der
> Taskleiste**. Dort (Rechtsklick) findest du alle Funktionen und Einstellungen.

## Systemvoraussetzungen

- **Windows 11** oder **macOS** (Apple Silicon empfohlen). Linux experimentell.
- Ein **Mikrofon**.
- ~1–2 GB freier Speicher (das GPU-Paket ist größer).
- Für das CUDA-Paket: eine **NVIDIA-Grafikkarte**.
